### Переменные

Есть 4 остновных типа данных:

1. int - целочисленный
2. float - числа с плавающей точкой
3. boolean - логический
4. string - строковый 
5. list - списки (о них позже)

#### Пару слов об интерполяции.

Недавно в Python была добавления *интерполяция*. Синтаксис у неё такой:

<code>
a=12

b=1.25

c='Nya'

print('{}-{}-{}'.format(a,b,c))
</code>

или

<code>
a=21

b=25.135

c='UwU'

print(f'{a}-{b}-{c}')
</code>
